== Description ==

Klout Social Profile Widget displays your Klout score together with you social media profiles (Facebook, Twitter, LinkedIn, etc) on your Wordpress Sidebar.

== Installation ==

1. Upload `KloutSocialProfile.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the 'Klout Social Profile Widget' to sidebar via the 'Appearance/Widgets' menu in Wordpress
